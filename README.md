Java Swing Application for Managing Persons and Their Subtypes

Overview

This project is a Java Swing application that allows users to create, edit, delete, save, and load Person objects and their subtypes. It includes full file-menu functionality, object serialization, custom date validation, and GUI-driven workflows that match the Final Project specification.

The application supports these object types:

Person

RegisteredPerson

OCCCPerson

Each entry may contain child elements and validated date fields (throwing and handling OCCCDateException as required).
